# Joke API 😄

A simple REST API that returns random jokes.  
Built using **Node.js** and **Express**.

---

## 🚀 Features
- Get random jokes
- Get jokes by category (e.g., Programming, General, Dad Jokes)
- Lightweight and fast
- Easy to extend with new jokes

---

## 🛠️ Tech Stack
- Node.js
- Express.js

---
